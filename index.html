Last.fm Now Playing Widget

A jQuery plugin for showing the currently playing track on Last.fm.

  • Team Friendly

    Find the most recently played track from multiple users

  • Easy to Customize

    Create your own markup to display the track information

Using Last.fm Now Playing Widget

Requirements

The plugin requires jQuery, we recommend using the latest version. You also need to have a Last.fm API key.

Installation

  1. Add Required Scripts

    Make sure you include jQuery before adding the plugin at the bottom of your HTML document

    <script src="/scripts/vendor/jquery.js"></script>
<script src="/scripts/plugins/lastfmnowplaying.js"></script>

  2. Add a Template

    Add a template into your HTML document as in the example below. This can appear anywhere on your web page and can contain any markup you'd like. The template makes use of a number of custom tags surrounded by {}'s. Each template must have it's own ID.

    <script type="text/template" id="lastFmWidget">
	<div>{ track.artist }</div>
	<div>{ track.title }</div>
	<div>{ track.album }</div>
	<img src="{ track.image.medium }">
</script>

    Here is a full list of available tags:

    { track.artist }
    The track's artist
    { track.title }
    The title of the track
    { track.album }
    The album the track belongs to
    { track.url }
    A URL to track information on Last.fm
    { track.image.small }
    A 34 x 34px image of the track's artwork
    { track.image.medium }
    A 64 x 64px image of the track's artwork
    { track.image.large }
    A 126 x 126px image of the track's artwork
    { track.image.extralarge }
    A 300 x 300px image of the track's artwork

  3. Call Widget

    The widget is called using the following in your main JavaScript file.

    $('#lastFmWidget').lastfmNowPlaying({
	apiKey: 'YOUR-API-KEY',
	members: ['YOUR-LASTFM-USERNAME']
});

    Be sure to update the apiKey and members options with your own information.

Template Examples

Here are a few examples of different templates for the plugin. You can write whatever code you like for the template, the only requirement is a unique ID for the template and your desired combination of custom tags.

  • Large Artwork Template

    <script type="text/template" id="largeArtworkTemplate">
  <a href="{ track.url }" class="large-widget" alt="View information on Last.fm">
    <div class="large-widget__image-wrapper">
      <img class="large-widget__image" src="{ track.image.extralarge }" alt="Album artwork">
    </div>
    <div class="large-widget__details">
      <div class="large-widget__artist">{ track.artist }</div>
      <div class="large-widget__title">{ track.title }</div>
    </div>
  </a>
</script>

  • Minimal Template

    <script type="text/template" id="minimalTemplate">
  <a class="minimal-widget" href="{ track.url }">
    <img class=" minimal-widget__image" src="{ track.image.small }">
    <div class="minimal-widget__details">
      <div class="minimal-widget__artist">{ track.artist }</div>
      <div class="minimal-widget__title">{ track.title }</div>
    </div>
  </a>
</script>

  • Kitchen Sink Template

    <script type="text/template" id="kitchenSinkTemplate">
  <div class="kitchen-sink-widget">
    <div class="kitchen-sink-widget__artist">Artist: { track.artist }</div>
    <div class="kitchen-sink-widget__title">Title: { track.title }</div>
    <div class="kitchen-sink-widget__album">Album: { track.album }</div>
    <div class="kitchen-sink-widget__url">URL: { track.url }</div>
    <img class="kitchen-sink-widget__image kitchen-sink-widget__image--small" src="{ track.image.small }">
    <img class="kitchen-sink-widget__image kitchen-sink-widget__image--medium" src="{ track.image.medium }">
    <img class="kitchen-sink-widget__image kitchen-sink-widget__image--large" src="{ track.image.large }">
    <img class="kitchen-sink-widget__image kitchen-sink-widget__image--extralarge" src="{ track.image.extralarge }">
  </div>
</script>
